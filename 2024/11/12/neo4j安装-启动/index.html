Neo4j 安装与启动教程

发表于 2024-11-12 更新于 2026-04-13 分类于 数据库

最近在做一个基于知识图谱的智能生成项目，需要用到 Neo4j 图数据库。本文记录 Neo4j 的安装及启动过程。

一、安装 Neo4j

1. 安装 JDK 并配置环境变量

Neo4j 是基于 Java 的图形数据库，运行时需要启动 JVM 进程，因此必须先安装 Java SE JDK。

从 Oracle 官网 下载 JDK，推荐使用 JDK 1.8，安装后配置好环境变量。

2. 下载 Neo4j

官方下载地址：https://neo4j.com/deployment-center/

下载完成后，Neo4j 的主要目录结构如下：

目录 说明
bin 存储 Neo4j 的可执行程序
conf 控制 Neo4j 启动的配置文件
data 存储核心数据库文件
plugins 存储 Neo4j 插件

3. 配置环境变量

新建系统环境变量 NEO4J_HOME，将 Neo4j 的安装目录作为变量值。

配置文件存储在 conf 目录下，默认情况下无需修改即可启动。

二、启动 Neo4j

1. 以管理员身份打开命令提示符

按 Win + R 输入 cmd，右键以管理员身份运行。切换到 Neo4j 安装目录。

注意：必须以管理员身份运行，否则命令无法识别。

2. 执行启动命令

命令行开头显示 C:\Windows\system32\... 即为管理员身份。

三、访问 Neo4j 浏览器

Neo4j 内置了一个 Web 浏览器界面，启动后访问：

  • 初始用户名：neo4j
  • 初始密码：neo4j（首次登录需要修改密码）
